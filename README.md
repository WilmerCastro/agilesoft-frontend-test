# AgileMovies

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.0.3.

## Pre-requisitos

* Node.js
* Angular CLI
* Git
* IDE (Visual Studio Code, Sublime Text, Atom, etc.)

## Instrucciones para ejecutar el proyecto

* Clonar el repositorio
* Ejecutar `npm i`
* Ejecutar `ng serve`
